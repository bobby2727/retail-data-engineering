# Retail Data Engineering Project

## Overview

This project demonstrates an end-to-end Data Engineering pipeline using Python, Pandas, PySpark, and Spark SQL.

## Technologies

- Python
- Pandas
- PySpark
- Spark SQL
- Parquet
- Git
- GitHub

## Project Structure

- Extract
- Validation
- Transformation
- Loading
- Spark SQL Analytics

## Features

- ETL Pipeline
- Data Validation
- Bronze, Silver, Gold Layers
- Spark DataFrames
- Spark SQL
- Analytical Queries

## How to Run

```bash
pip install -r requirements.txt
python main.py
python spark_sql.py
```
